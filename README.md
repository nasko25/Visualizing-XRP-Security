# To build and run the docker image:
```
docker build . -t <image_name>
docker run -p 3000:3000 -p 8080:8080 -d <image_name>
```
Or using `docker-compose`:
```
docker-compose build
docker-compose up -d
```
after that you can:
```
docker-compose start/stop/restart
```
